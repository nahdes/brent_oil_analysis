# Data Directory

## Structure

- `raw/` - Original, immutable data files (Brent oil prices CSV)
- `processed/` - Cleaned and processed data ready for analysis
- `events/` - Major geopolitical and economic events dataset
- `external/` - Any additional external data sources

## Data Sources

- Brent Oil Prices: Daily prices from May 1987 to November 2022
- Events Data: Compiled list of major market-moving events
